## Nginx (reverse proxy)
- Чтобы запустить проект у вас должен быть установлен Docker и Docker compose.Запускается же всё одной командой:
``` docker compose up -d ```

- Для проверки можно сначала убедиться что контейнеры запущены командой:
``` docker ps ```.  Мы увидим 2 запущенных контейнера и далее обращаемся к веб-приложению командой:
``` curl http://localhost ```. После чего мы увидим ответ от нашего веб-приложения, а именно "Hello from Effective Mobile!"

- схема (nginx → backend) работает так:
nginx стоит перед нашим основным приложением и принимает все запросы от пользователей, а затем перенаправляет их на backend.
